Les landmarks

L'attribut role propose un ensemble de valeurs permettant de définir les grandes structures de la page.

Ces repères peuvent servir de point de navigation (de manière similaire aux titres) dans la structure pour naviguer rapidement.

Démonstration

Avec NVDA, visiter le site Alsacréations.

Vous avez deux manières d'utiliser les landmarks.

  1. Utiliser la touche d. Chaque pression sur la touche vous permettra d'atteindre les repères les uns après les autres.
  2. Utiliser le raccourci NVDA + F7 et choisir « repère ». La liste des repères apparaît : sélectionner un repère pour l'atteindre.

Les repères sur cette page

Sur cette page, quatre repères ont été implémentés :

  • role="banner" englobe le logo et le titre ;
  • role="navigation" englobe le menu de navigation ;
  • role="main" englobe la partie centrale du contenu ;
  • role="contentinfo" englobe le pied de page.
