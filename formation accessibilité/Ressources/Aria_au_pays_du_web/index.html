Présentation

Ces pages contiennent les démonstrations effectuées lors de l'atelier « Aria au pays du web », le 16 avril 2011, lors du W3Café Accessibilité.

Les démonstrations présentées ici sont volontairement très simplifiées, le but étant d'illustrer certains concepts de base d'ARIA.

Attention !

  • Il s'agit d'une simple démonstration. Le support sur d'autres navigateurs et d'autres lecteurs d'écran n'est pas garanti. Néanmoins, les mêmes démonstrations ont fonctionné avec Jaws 11 dans la même configuration.
  • Seules les fonctionnalités de base ont été utilisées avec NVDA dans sa configuration par défaut.
  • ARIA est encore en phase d'expérimentation et il est possible que certaines démonstrations fonctionnent mal selon votre configuration.
